# 🚀 AI Model Dashboard

### 🧠 Overview
This is a full-stack web application built with **React**, **Framer Motion**, **GSAP**, and **Lenis** on the frontend, and **Node.js + Express + MongoDB + Firebase Admin** on the backend.  
The app allows users to view, manage, and store AI model–related data seamlessly.

## 🧩 Tech Stack

### **Frontend**
- ⚛️ React.js (Vite)
- 🎞 Framer Motion — smooth UI animations
- 💫 GSAP (GreenSock) — advanced scroll & reveal effects
- 🌀 Lenis — smooth scroll experience
- 🎨 Tailwind CSS — responsive design

### **Backend**
- 🚀 Express.js
- 🗃 MongoDB (Atlas)
- 🔥 Firebase Admin SDK (Firestore)
- 🔐 Dotenv for environment variables
- 🌍 CORS enabled for frontend connection
